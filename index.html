Wireframing Application

Freelancer.com
In-house Developer

Built a high fidelity wireframing tool to help Product Managers, Designers and Developers create interactive web application mockups. The application allows anyone to test their ideas rapidly and interact with the web page as though the feature has already been built.

The application pulls and renders actual web components from Freelancer’s design system and can easily be customized using form controls.

Angular, TypeScript, RxJS, HTML, SCSS/CSS, Node.js, etc.
